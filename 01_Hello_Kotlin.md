# Lesson 01 - Hello Kotlin

Goal:
Write and run your first Kotlin program.

## First Program

Create Main.kt and write:

fun main() {
    println("Hello, Kotlin!")
}

What this means:
- fun defines a function.
- main is the program entry point.
- println prints text to output.

## Kotlin File Basics

- .kt files store Kotlin source code.
- One file can have many functions and classes.
- Kotlin uses braces for code blocks.

## Comments

Single line:
// This is a comment

Multi-line:
/*
This is a
multi-line comment
*/

## Exercise

1. Print your name.
2. Print your goal as a developer.
3. Print the current year.

Example:

fun main() {
    println("My name is Alex")
    println("Goal: Become a Kotlin mobile developer")
    println("Year: 2026")
}

Create output using 5 separate println lines.
