# Kotlin Beginner Tutorial

This folder contains a complete Kotlin 101 training path for beginners.

Who this is for:
- New developers with little or no Kotlin experience
- Mobile developers onboarding to Kotlin-based Android projects

How to use this tutorial:
1. Start with 00_Apps_And_Setup.md
2. Follow lessons in order from 01 to 10
3. Complete exercises in each lesson before moving on
4. Build one mini project in 09_Practice_Projects.md

Suggested schedule:
- Day 1: Setup + Lessons 01-03
- Day 2: Lessons 04-06
- Day 3: Lessons 07-08
- Day 4: Lesson 09 project build
- Day 5: Review with 10_Cheat_Sheet.md
