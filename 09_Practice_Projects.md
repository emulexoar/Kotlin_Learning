# Lesson 09 - Practice Projects

Goal:
Apply everything by building small beginner projects.

## Project A - Console Student Grade Tracker

Features:
1. Input student names and scores
2. Compute average
3. Print pass/fail per student
4. Print top scorer

Concepts used:
- Variables
- Lists
- Functions
- Control flow
- Data classes

## Project B - Simple To-Do App Logic (No UI required)

Features:
1. Add task
2. Remove task
3. Mark done
4. Show pending vs done counts

Concepts used:
- Mutable list
- Data class
- Functions
- Filter/map/count

## Project C - Currency Converter (Console)

Features:
1. Input amount
2. Select source and target currency
3. Convert using hardcoded rates
4. Handle invalid inputs safely

Concepts used:
- when
- null safety
- functions
- number formatting

## Delivery Requirement

For at least one project, submit:
1. Source files
2. Sample input/output
3. Short README with what you learned

## Stretch Goal

Use coroutines to simulate API latency in one project.
